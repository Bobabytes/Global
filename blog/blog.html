My Todo List

Author: Ross & Riley.

Published on: November 11, 2023

  1. Theme Selection:
    • ☑ Choose a theme of your choice for your webpage.
  2. HTML Usage:
    • ☑ Create a well-structured webpage using HTML.
    • ☑ Include headings, paragraphs, articles, links, images, buttons, lists, tables and a video from the web.
    • ☐ Maybe stop spending so much time on HTML.
  3. CSS Styling:
    • ☑ Implement CSS to style your webpage. You should:
      • ✓ Change text color, use different fonts.
      • ✓ Add background colors/ images, borders.
      • ✓ Use margin, padding, and outline boxes.
      • ✓ Ensure that your design is consistent and visually appealing.
  4. Responsive Design:
    • ☑ Make your webpage fully responsive to different devices using media queries.
  5. Navigation Bar:
      ☑ Create a functional navigation bar with at least 4 menu items.
    • ☑ Ensure that menu items link to the relevant sections of your webpage.
  6. Contact form:
    • ☑ Include a contact form in your webpage. The form should:
      • ✓ Have fields for Name, Email, Telephone Number, and a Message.
      • ✓ Implement HTML validation to check for a valid email format and a specific telephone number length.
      • ✓ The “Message” field should have a maximum character limit of 200 characters. Implement JavaScript to count and display the number of characters entered in the “Message” field in real-time.
  7. JavaScript Interaction:
    • ☑ Use jQuery to create interactions on your webpage. You should:
      • ✓ Implement alerts to provide feedback to users.
      • Add a onmouseover event that triggers changes in the interface.
      • Use a onkeydown event to create interactivity when a key is pressed.
  8. Content Quality:
    • ☑ Ensure the content on your webpage is high-quality, relevant to your chosen theme, and well-written with proper spelling and grammar.
  9. Creativity and Originality:
    • ☑ Add creative and original elements to your webpage's design or content to make it unique.
  10. Documentation and Code Organization:
    • ☑ Keep your project files well-organized.
    • ☑ Provide clear documentation within your code using comments to explain its structure and functionality.
